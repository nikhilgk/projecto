projecto
========

Angular-Node-Mongo-Postgres-Heroku seed project

### Install node.js from source:
	git clone git://github.com/joyent/node.git
	v0.10.3
	cd node/
	./configure 
	make
	sudo make install


### Copy and setup the project:
	git clone https://github.com/nikhilgk/projecto.git
	cd projecto 
	npm install
